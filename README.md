### Current mission

Solar companies still run too much of their business through spreadsheets, chat threads, and software that does not talk to each other.

I’m building Solar1 to connect sales, projects, permits, field work, inventory, finance, and the people behind them.

One business. One system. Fewer things falling through the cracks.

### How I build

```text
find a painful problem
        ↓
talk to the people living with it
        ↓
build the smallest useful version
        ↓
break it
        ↓
fix it
        ↓
repeat until boringly useful
```
